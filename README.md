Project Overview:
-----------------
This project is a fully responsive personal portfolio website developed using
HTML5, CSS3, Bootstrap 5, and JavaScript. The website showcases personal
information, technical skills, academic projects, and contact details in a
professional and visually attractive format.

Features:
---------
- Responsive design for desktop, tablet, and mobile devices
- Fixed navigation bar with smooth scrolling
- Animated hero section with profile image
- Skills section with interactive cards
- Projects section with GitHub and live demo links
- Contact section with LinkedIn, Gmail, GitHub, Instagram, and phone number
- Dark theme with neon green styling
- Hosted online using GitHub Pages

-----------

2. Add your profile photo and rename it to profile.jpg.
3. Open index.html in a browser.
4. Customize links and content as needed.
